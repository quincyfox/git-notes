## Git Setup: Linux

- `Git` already comes pre-installed in Linux.

- To check `Git` version:

```shell
$ git --version
```

- To update `Git` for Debian/Ubuntu systems:

```shell
$ sudo apt-get update
$ sudo apt-get install git
```

- To update `Git` for Fedora systems:

```shell
$ sudo dnf install git
```
or
```shell
$ sudo yum install git
```
